# Pheglovog's Blog

这是一个基于 Hugo 构建的个人博客，专注于量化交易、区块链和全栈开发。

## 项目结构

- `content/` - 博客文章和页面内容
- `layouts/` - Hugo 模板文件
- `static/` - 静态资源（CSS、JS、图片）
- `data/` - 数据文件
- `config.toml` - Hugo 配置文件

## 开发

```bash
# 安装 Hugo
brew install hugo

# 本地开发
hugo server -D

# 构建
hugo

# 部署到 GitHub Pages
hugo --baseURL https://username.github.io/repo --cleanDestinationDir
```

## 功能特性

- ✅ SEO 优化（Open Graph、Twitter Card）
- ✅ 响应式设计
- ✅ RSS 订阅
- ✅ Sitemap 自动生成
- ✅ 极简设计风格
